Hassle-free Automation

Leveraging the popularity and the workflow of git, we built a system that would seamlessly fit the developer workflow.

In its initial form, git://ci is built around the work Linaro is doing with the zephyr project.

With git://ci, you just “git clone” the source code, make your changes, and “git push” to a git://ci remote. Your code will be rebuilt - along with over 50 of the zephyr test binaries - and it will be tested using git://ci's machine farm currently running qemu cortex-m3 virtual machines.
