Using git://ci is simple:

  1. Obtain access
  2. Clone zephyr source code
  3. Add gitci remote
  4. Review test results

Obtain access

  1. Obtain access and upload your SSH key.
  2. Get your username and repository URL.

Obtain source code

  • $ git clone https://gerrit.zephyrproject.org/r/zephyr zephyr-project
  • $ cd zephyr-project
  • Modify your source code, making sure it builds locally. 
  • $ git add .
  • $ git commit -m "Commit message"

Add git://ci remote

  • $ git remote add gitci URL

Enable git://ci

  • $ git push gitci master
  • Lives results will populate your screen…

Access your results

  • The system will store your results and provide you with a link.
